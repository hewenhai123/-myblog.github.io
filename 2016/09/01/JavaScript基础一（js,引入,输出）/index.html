js基础一

发表于 2016-09-01   |   分类于 js ， 基础

主要记录关于：页面的组成部分、引入到页面中的几种方式、js的几种输出方式、js的组成、变量、命名规范

页面的组成部分:

  1. HTML –> 超文本编辑语言 XHTML–>更严谨的HTML–> DTHML –>动态展示的HTML;
  2. CSS –> 层叠样式表;
  3. JavaScript –>轻量级的脚本编程语言

js引入到页面中的几种方式：

  1. 行内样式：和css一样写在行内（强烈不推荐，有安全问题）;
  2. 内嵌式：;
  3. 外链式：(标签内不能再编写代码，不生效);

js的几种输出方式：

  1. alert(“ 欢迎”) –>在页面中弹出一个框里面展示内容 ==（会把输出的内容转换成字符串在输出）==;
  2. document.write(“欢迎”) –>在文档中直接输出内容 ==（会把输出的内容转换成字符串在输出）==;
  3. console.log() –>在控制台输出日志;
  4. console.dir();
  5. console.table;
  6. innerHTML:获取内容或者指定对象插入内容 ==（可以插入HTML结构）==;
  7. innerText: 获取内容或者指定对象插入内容 ==（会自动去除THML结构）==;

js的组成：

  1. js是由三部分组成;
  • ECMAScript(4.0~5.0):定义了JS中的，语法、命名规范、变量、数据类型、操作语句等最为核心的基础;
  • DOM: document object model 文档模型文档对象模型 提供了我们操作当前页面中元素标签所有的属性和方法;
    document :文档对象;
  • BOM: browser object model 浏览器对象模型 提供了js操作浏览器的所有属性和方法
    window: 浏览器对象;
  1. js 变量;
  • 变量（可变的量）一个名字，用来存储值的名字;
  • 常量（不可变量） 固定的值;
  • JS中定义变量方式： var 名字(变量名字) = 值 var num=12
  1. js 命名规范:
  • JS 中严格区分大小写;
  • 命名必须遵循”驼峰命名法”;
  • 不能使用关键字和保留字;
  • 关键字：在JS 中有特殊的用法的词 例如 var;
    break case catch continue default delete do else finally for if in function var switch;
  • 保留字：未来可能会成为关键字的;
    boolean byte char class int float goto long short import;
