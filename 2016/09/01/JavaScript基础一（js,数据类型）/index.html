js基础一之数据类型

发表于 2016-09-01   |   分类于 js ， 数据类型

主要记录关于：js的数据类型、基本数据类型、引入数据类型、基本数据类型和引用数据类型的区别

js的数据类型

  1. 前端的变量是松散类型的，不管什么样的数据类型定义变量都用 var
  2. JS中的数据类型分为：
  • 基本数据类型（值类型）：

number、string、boolean、null、undefined;

  • 引用数据类型：

    对象数据类型：

    {}、[]、/^$/……

    函数数据类型：

    function(){};

  1. JS中的数据类型检测方式：返回的是字符串

  • typeof –>用来检测数据类型的运算符,检测对象数据类型返回值都是”object”

    局限性：不能具体细分对象数据类型的值，

    typeof value typeof 12 –>”number” ;

    typeof null —> “object”; 因为null是空对象指针

  • instanceof;

  • constructor;
  • Object.prototype.toString.call();

基本数据类型 number：

number ： 正数、负数、零、小数、NaN;

  1. NaN: not a number 不是一个数，但是它属于number 数据类型的 typeof NaN –>”number”;
  2. NaN和NaN本身是不想等的; NaN==NaN–> false
  3. 什么情况下出现NaN;
  • 数字+undefined=NaN;
  • 在无效的数学运算中;
  • 把其他的数据类型转换为number类型的时候不能正常转换的，结果都是NaN
  1. 如何把其他的数据类型转换为number；
  • Number (value):强制严格转换，值中不能包含非数字的字符，否则值为NaN
  • parseInt / parseFloat（可识别小数点）:非严格转换从左到右一个一个的转换，只要遇到一个非有效的,则停止查找,不管后面是否还有数字都不在查找了
  • isNaN(value)：检测一个值是否为有效数字，如果是有效数字返回false，如果不是有效数字返回true

基本数据类型 boolean：

布尔：只有 true false 两个值;

  1. 将其他的数据类型转换布尔类型只有：0 、NaN、null、undefined、””;
  2. 如何将其他的数据类型转换成布尔类型;
  • Boolean() 强制转换；
  • ! 先把其转换为布尔后让后再取反;
  • !! 转换布尔类型;
  • 在条件中判断 if(1==1){} 首先判断1和1 是否相等，相等返回true，否则返回false；
    if(1){} 如果条件中不是比较表达式也不是其他的东西，只是一个单独的值，首先把这个值转换为布尔类型，通过真假判断条件是否成立;
  1. null、undefined 的区别;
  • 两个都带都代表没有，但是有区别；
  • null是现在没有以后会有，意料之中的没有;
  • undefined 是意料之外的没有；

基本数据类型 string：

定义： 在js中只要是用 “ “/ ‘ ‘ 包起来的都是字符串，不管里面内容是什么；

  1. 每一个字符串都是有零到多个字符组成的;
  • 每一个字符串都有自己的位置 “索引”;
  • 索引是从零开始的 str[索引];
  1. 加法在JS中不一定是数学运算，和字符串想加时属于字符串拼接;
  2. 乘、除、减、遇到字符串也是数学运算，把字符串转换为数字进行操作;

引用数据类型 对象：

  1. 对象数据类型都是由属性名和属性值组成的；
    一个对象是由多个属性名和属性值组成的（多组键值对 key：value）;

  2. 给已给对象增加/修改属性名和属性值：重要：JS规定一个对象中的属性名是不能够重复的，如果重复了，按照最后的一个对应的值为主;

  3. 增
    obj.name=”名字” obj[“name”]=”名字”

  4. 修
    和增加一样如果原对象中存在相同的属性名，则修改，如果不存在则添加；
    obj.name=”名字” obj[“name”]=”名字”

  5. 获取：
    obj.name obj[“name”]

  6. 删除：
    假删除：没有把属性删除，只是把属性值变为空;
    obj.name=null —->obj[“name”]=null
    真删除：
    delete obj.name; —->delete obj[“name”]

  7. 重点记忆：
  • 一个对象中的属性值可以是纯数字
  • 我们必须这样操作 obj[“7”]—>obj[7]
  • 引号可加可不加，非数字的必须加””

基本数据类型和引用数据类型的区别：

  1. 基本数据类型也叫做值类型，是直接操作值的;
  2. 引用数据类型操作的是对内存空间的引用地址
  • 浏览器开始加载我们的HTML页面的时候，首先或形成一个”全局作用域”共js代码执行
  • JS从上之下开始执行;
  • 引用数据类型的赋值操作基本数据类型第不一样的;

a 开空间：遇到对象或者function时首先开一个新内存空间；

b 分地址：浏览器给开辟的内存空间一个地址

c 存数据：对象会将大括号中的内存错储，function会以字符串的形式存储

d 在赋值：吧内存地址赋值给变量名;
